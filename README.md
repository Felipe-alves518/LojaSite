<h1>Loja-Site</h1>

Tecnologias usadas: React, Vite, MongoDB, Node.Js, express

Descrição: O projeto visa simular um site de uma loja, no qual apresenta um banco de dados (MongoDB) que armazena informações de login e compras. A comunicação com o banco de dados ocorre com uma Api feita com Node.Js e express
